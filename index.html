GourmetGo

This project is aimed at developing a web based application that depicts the online ordering of food. The key intent is to make it simple for individuals to access a wide range of dishes from various restaurants.This food delivery application uses MongoDB database to manage customer interactions, restaurant operations, orders, and payments. With this, users can seamlessly create accounts and login to explore restaurants, view menus, and enjoy the convenience of home delivery or convenient pick-up options. It also allows users to keep track of their past orders and review delivered items. It offers a convenient and contemporary delicious experience because of its user-friendly interface and extensive restaurant connections, catering to the changing preferences and interests of customers.

Leveraging AWS Infrastructure For Seamless Medical Services

The project involves creating a cloud-based hospital medical appointment system, leveraging AWS infrastructure to support a highly available, scalable, and secure platform. This system is designed to facilitate efficient appointment scheduling, allowing doctors to allocate 15-minute slots to each patient. Doctors can customize their schedules, prescribe medications, and patients can provide feedback directly through the platform. The integration of a custom-built payment management system ensures a seamless financial transaction process for services rendered.The project will employ Docker for application containerization, GitHub for source code management, Github projects for project tracking, Postman for API testing, Flask for backend development, and MongoDB for database management. These technologies complement the AWS services, creating a robust infrastructure for the hospital appointment system.

Crop Monitoring using Wifi controlled Rover

This rover's duties include exploring the fields, providing video data from the surrounding area, and taking pictures of the tomato crops. We analyze the collected images in the cloud using a set of built-in ML algorithms to ascertain the state of the crops. This rover also makes use of a variety of sensors to gather data and transmit it to the microcontroller, which regulates the behaviour of the rover. The proposed work takes into account a wifi-controlled rover that can check any kind of areas and can analyze the surrounding conditions with the aid of the camera with high speed and precision. A wirelessly operated two-wheeled robotic rover is built using an Arduino Uno gadget coupled with a Wi-Fi module. This Rover has the ability to actively engage with its surroundings and can be remotely operated for inspection reasons. Technologies used : Internet of Things(IoT), Machine Learning(ML) Tools used : Arduino Uno, Esp32 camera module, motor drivers(L298N), Arduino IDE
